# SneakerHead 👟

SneakerHead is a real-time sneaker auction platform where users can buy, sell, and bid on sneakers through live auctions.

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB (Planned)

## Project Structure

frontend/ - React frontend

backend/ - Express backend

docs/ - Project documentation

## Current Progress
- React frontend setup using Vite
- Express backend setup
- Middleware configuration
- Health check API endpoint
- GitHub repository setup
